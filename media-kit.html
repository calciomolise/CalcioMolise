Processo collaborazione

Semplice, tracciabile, senza giri.

Step

  1. Richiesta → DM o email
  2. Pacchetto → livello presenza + calendario
  3. Asset → logo + link + brief (2 righe)
  4. Pubblicazione → output programmati
  5. Report → screenshot + riepilogo

Asset richiesti

  • Logo (PNG trasparente o SVG)
  • Link / username
  • Descrizione brand (2 righe)
  • Preferenze: sobrio / informativo
